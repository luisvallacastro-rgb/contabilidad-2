# Contabilidad II

Base interactiva organizada desde el documento definitivo `Matriz de Planeacion de clases.xlsx`:

- 80 clases.
- 20 semanas.
- 4 unidades valorativas.
- 4 unidades tematicas y cierre integrador, sin bloque introductorio separado.
- Enfoque en NIIF para PYMES, referencia NIIF completas, estrategia didactica y caso numerico integrador semanal.

## Estructura

- `index.html`: portada, mapa academico y matriz de 80 clases con buscador.
- Seccion `Evaluacion por semana`: 20 bloques semanales con ejercicio practico aplicado.
- `modulo-0-introduccion.html`: clases 01-04.
- `modulo-1-estados-financieros.html`: clases 05-20.
- `modulo-2-efectivo-bancos.html`: clases 21-40.
- `modulo-3-inversiones-cxc-iva.html`: clases 41-60.
- `modulo-4-inventarios-anticipados.html`: clases 61-77.
- `cierre.html`: clases 78-80.
- `module-pages.css` y `module-pages.js`: estilos, matriz e interacciones compartidas.

## Enfoque semanal

Cada semana agrupa cuatro clases. Las clases siguen siendo medibles de forma individual, pero la evidencia principal del aprendizaje es un ejercicio semanal con:

- Enunciado practico.
- Desarrollo sugerido.
- Puntaje semanal.
- Integracion con la unidad y la base tecnica correspondiente.
